esp_callback_t conn,
esp_callback_t disconn,
callback_data_t data_received )

Configura e inicia la conexión al servidor MQTT.

Esta función establece la conexión inicial al servidor MQTT especificado por uri. Se pueden proporcionar callbacks para manejar eventos de conexión, desconexión y recepción de datos.

Parameters
uriURL del servidor MQTT.
connFunción de callback llamada al conectarse al servidor.
disconnFunción de callback llamada al desconectarse del servidor.
data_receivedFunción de callback llamada al recibir datos desde un canal subscripto.
Returns
esp_err_t ESP_OK si la operación se realizó correctamente, de lo contrario un código de error.
