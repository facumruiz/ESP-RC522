#define FREE ( ptr)     if(ptr) { free(ptr); ptr = NULL; }

Macro for safely freeing memory. This macro checks if the pointer is not NULL before calling the free function. After freeing the memory, it sets the pointer to NULL to prevent dangling pointer issues. This helps in avoiding double free errors and improves the safety of memory management.

Parameters
ptrPointer to the memory to be freed.
